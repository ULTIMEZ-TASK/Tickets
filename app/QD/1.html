v1.0.0

Sep 01, 2024

Initial release with basic navigation and back arrow functionality.

  • Maximum 6 breadcrumb levels allowed for standard viewports to avoid UI clutter.
  • Labels must be plain text and should not exceed 20 characters per item.
  • Component must include `aria-label="Breadcrumb"` for screen reader compatibility.
  • Back button must always have a valid `href` or `onClick` handler.

Scenario 1: Truncation & Tooltip

Verify long project titles truncate with "..." on mobile widths and show full title on hover via tooltip.

Scenario 2: Keyboard Navigation

Ensure users can navigate through breadcrumb links using the "Tab" key and activate them with "Enter".

Scenario 3: Empty Breadcrumbs

Component should gracefully hide or show only the 'Home' link if no items array is provided.
