# CoinPedia FactCheck Dashboard

Live dashboard for CoinPedia cryptocurrency price prediction articles.

## Files

- `index.html` — the dashboard (password protected)
- `data.json`  — article data exported from BigQuery

## Updating Data

1. Run Cell 11 in the Colab notebook after each batch
2. Download the new `data.json` from Colab
3. Upload it to this repo (replace the old one)
4. Dashboard updates automatically

## Password

Set in `index.html` — search for `DASHBOARD_PASSWORD`
